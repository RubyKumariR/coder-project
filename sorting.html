Sorting Algorithms

Bubble Sort

Best Case: O(n)

Worst Case: O(n²)

Bubble Sort is a simple sorting algorithm that repeatedly steps through the list, compares adjacent elements, and swaps them if they are in the wrong order.

Selection Sort

Best Case: O(n²)

Worst Case: O(n²)

Selection Sort is an in-place comparison sorting algorithm. It divides the input list into two parts: the sublist of items already sorted and the sublist of items remaining to be sorted.

Insertion Sort

Best Case: O(n)

Worst Case: O(n²)

Insertion Sort is a simple sorting algorithm that builds the final sorted array one item at a time.

Merge Sort

Best Case: O(n log n)

Worst Case: O(n log n)

Merge Sort is an efficient, stable, comparison-based sorting algorithm. It works by dividing the unsorted list into n sublists, each containing one element.

Quick Sort

Best Case: O(n log n)

Worst Case: O(n²)

Quick Sort is an efficient, in-place, comparison-based sorting algorithm. It works by selecting a 'pivot' element from the array and partitioning the other elements into two sub-arrays.
